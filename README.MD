百度随心听 for BlackBerry 10
===========================

2016.7.7 本应用全面免费，开源的代码中已经注释掉了“解锁”相关的代码。

本项目使用[BlackBerry WebWorks](https://developer.blackberry.com/html5/) 工具编译，界面部分用了 [bbui.js](https://github.com/blackberry/bbUI.js) 来展现原生样式。本项目中使用的插件清单如下：

## HOW TO BUILD

由于webworks / cordova带有大量框架内容，本repo内的内容仅为项目内容，clone到本地后，需要使用webworks create创建项目，手动添加上面所列插件，然后在项目目录下使用`webworks build`等命令进行编译、调试等，有关webworks工具的使用说明，请参见[BlackBerry WebWorks](https://developer.blackberry.com/html5/) 中的相关内容。
